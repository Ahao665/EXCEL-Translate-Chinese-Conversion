## Excel 全表翻译工具（测试版）

这是一个基于 **Vue 3 + Vite + Element Plus + SheetJS(xlsx)** 的前端小工具，用来对整个 Excel 工作簿进行「一键翻译」预览和导出。

当前仓库处于 **测试版本（Prototype）** 阶段，功能和接口可能随时变更，仅用于个人试验和演示，不适合作为生产环境使用。

### 主要特性（测试版）

- 上传 Excel 后，自动读取全部工作表。
- 一键「翻译全部」：对每个单元格做语言检测，并尝试统一翻译为中文。
- 中间区域可切换查看：
  - 原始数据：按“第几页”浏览原始工作表；
  - 处理结果（翻译后）：查看对应页的翻译结果。
- 导出时按 **原页 / 翻译页交错** 的方式输出，例如：
  - 第 1 页：原始 Sheet1  
  - 第 2 页：Sheet1 的翻译结果（`Sheet1_翻译`）  
  - 第 3 页：原始 Sheet2  
  - 第 4 页：Sheet2 的翻译结果，以此类推。

### 本地运行（开发模式）

```bash
cd frontend
npm install
npm run dev
```

在浏览器中打开终端输出的本地地址即可访问测试版页面。
